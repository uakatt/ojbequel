- (SpringLoad) initialize(spring_bootstrap_file, options = {})

Create a new OJBequel::RepoFiles::SpringLoad to maintain a hash of OJB repository files, in (name : path) Hash format.

Examples:

Save names of OJB files like "ojb-foo.xml"

OJBequel::RepoFiles::FileGlob.new('/java/projects/kc/src/main/resources/kc-bootstrap-springbeans.xml')

Parameters:

  • spring_bootstrap_file (String) —

    the Spring bootstrap file (either a properties file or an XML file) of the project that uses OJB. This file is expected to point to Spring XML files that each reference one or more OJB repository files.

  • options (Hash) (defaults to: {}) —

    various options

Options Hash (options):

  • :type (Symbol) —

    If spring_bootstrap_file ends in ".xml" or ".properties", the file type will be figured out automatically. Otherwise, you can specify :type => :xml or :type => :properties.

def initialize(spring_bootstrap_file, options={})
  @spring_bootstrap_file = spring_bootstrap_file
  if @spring_bootstrap_file =~ /\.xml$/
    @type = :xml
  elsif @spring_bootstrap_file =~ /\.properties$/
    @type = :properties
  end
end
