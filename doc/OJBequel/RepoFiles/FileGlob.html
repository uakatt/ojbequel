Overview

Use FileGlob if the OJB repository files can simply be gathered by matching file globs. If the order of the OJB repository files matters (as it often does, like if one repository overrides the <class-definition>'s of another), then a different class, like SpringLoad, might be more useful.

- (FileGlob) initialize(base_dir, file_re)

Create a new OJBequel::RepoFiles::FileGlob to maintain a hash of OJB repository files, in (name : path) Hash format.

Examples:

Save names of OJB files like "ojb-foo.xml"

OJBequel::RepoFiles::FileGlob.new('/java/projects/kfs', /ojb-(.+?).xml$/

Parameters:

  • base_dir (String) —

    the base directory of the project that uses OJB

  • file_re (RegExp) —

    the regular expression that will match an OJB repository file; it must contain at least one active group, which will be used for the name of the file (the key to the Hash).

def find
  @files = {}
  Find.find(@base_dir) do |path|
    next if path =~ /\.svn/
    if path =~ @file_re
      raise StandardError if $1 == nil
      @files[$1] = path
    end
  end
  @files
end
